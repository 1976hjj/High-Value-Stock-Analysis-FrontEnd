# 衡栖 · A 股多行业价值与危机防御前端

React + TypeScript + Vite 的多行业防御价值工作台。覆盖电信运营商、大型水电、优质银行、成熟收费公路、核电、综合油气、煤炭金属资源和消费龙头，并为各行业提供独立的指标、估值锚、风险闸门与情景传导逻辑。

## 启动

```powershell
.\install.cmd
.\dev.cmd
```

默认打开 `http://127.0.0.1:5173`。开发服务器将 `/api` 自动转发到 `http://127.0.0.1:8000`，因此请先启动后端：

```powershell
uvicorn bank_valuation.app.main:app --reload
```

若前后端分别部署，在前端环境变量中设置：

```powershell
$env:VITE_API_BASE_URL = "https://your-api.example.com"
.\dev.cmd
```

## 使用方式

顶部先选择行业，再选择对应龙头公司和估值日期。当前股票池覆盖 8 个行业、35 只核心公司；行业和公司清单由后端 `GET /api/strategy/universe` 返回，前端的默认目录与后端保持一致。

银行行业保留现有深度接口：

- `POST /api/bank/valuation`
- `POST /api/bank/monte-carlo`

并展示当前价格、PB 历史分位、PB-ROE/股息率/剩余收益模型、四情景价格带、风险标签与蒙特卡洛分位结果。电信、水电、收费公路、核电、油气、资源和消费行业调用 `POST /api/industry/analysis`，由后端返回估值分位、收益/防御/质量/风险综合评分、市场指标和风险提示。各行业页面保留不同的研究框架，例如电信关注 ARPU 和资本开支，水电关注来水与电价，收费公路关注剩余收费期，资源行业关注成本曲线和中周期现金流。

“数据全景”页进一步呈现全历史 PB / PE（如后端提供）/收盘价采样轨迹、3/5/10 年 PB 分位、估值使用的财务与风险输入、权益成本假设，以及 3/5/10 年剩余收益模型结果。

该页还会在主估值完成后后台加载 A 股银行横向基准：PB、PE、年化 ROE 和净利润同比的行业均值、中位数、P25–P75 区间及本行所在百分位。首次按估值日期汇总可能耗时较长，完成后后端按日期写入 CSV，并在 24 小时内优先读取。

策略回测支持单行业、多行业和全行业股票池，并包含等权/风险平价/评分加权、单行业权重上限、持仓数量和危机缓冲仓。银行单行业继续调用 `POST /api/bank/strategy-backtest`；包含非银行行业时调用 `POST /api/strategy/backtest`。点击“准备数据”会调用 `POST /api/strategy/backtest/prepare`，强制更新所选股票的历史数据缓存后执行同一套回测，因此首次准备全行业数据可能需要较长时间。

通用分析和回测中的行情、PB、PE、波动率与回撤来自后端历史数据；公司防御、分红、质量及股息率先验是当前研究先验，不是逐期财报事实，也没有消除核心股票池的幸存者偏差。所有内容为估值分析和风险提示，不构成投资建议。

## 联调与测试

先在后端启动 `uvicorn`，再运行前端开发服务器，即可通过 Vite 的 `/api` 代理完成联调。前端组件交互和 API 路由测试使用 Vitest + Testing Library：

```powershell
npm test
npm run build
```

`npm test` 覆盖行业选项卡、单/多行业股票池交互及银行/通用 API 分流；`npm run build` 同时执行 TypeScript 检查和生产构建。
